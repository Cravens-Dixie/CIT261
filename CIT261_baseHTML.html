Topic 9: Standard JS Events

Standard Events

Syntax: element event(event type)="some JavaScript"
For example, a button can have an onclick event that equals a JS Date()function.
A header can have onmouseover="style.color='red'" onmouseout="style.color='black'"
document.getElementById("myFrame").onload = function() {frameFunction()}
x.addEventListener("animationstart", myStartFunction)
JS functions are often called as event attributes

Other elements can have events, like this h4 element:

There are many many possible events.

All event objects are based on THE EVENT OBJECT and inherit all of its properties and methods. Some example events are Animation Event, Mouse Event, UiEvent, Drag Event, Focus Event, etc. These not only inherit from THE EVENT object, but they also have their own unique properties, methods, and event types.

Mobile Events

TouchEvents are related to UiEvents (User Interface)
AN example would be a body tag with property 'ontouchstart="isKeyPressed(event)'. The ontouchstart is an event type of TouchEvent, that calls the function isKeyPressed().
